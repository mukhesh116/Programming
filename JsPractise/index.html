  4. JavaScript Data Types:

    JavaScript provides different datatype to hold different values on variable. JavaScript is a dynamic programming language, it means do not need to specify the type of variable. There are two types of data types in JavaScript.

    • Primitive data type
    • Non-primitive (reference) data type
    Examples:

    // It store string data type

    var txt = "Mukhesh";

    var a = 5;
